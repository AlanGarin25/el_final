# Sistema Administrativo

Sistema de gestión con control de roles y versiones desarrollado con React.

## Características

- **Autenticación**: Sistema de login con validación de credenciales
- **Control de Roles**: Soporte para 3 niveles de usuario (Superusuario, Administrador, Usuario)
- **Dashboard**: Panel con estadísticas y gráficos en tiempo real
- **Notificaciones**: Sistema de notificaciones en tiempo real
- **Responsive**: Diseño adaptable a diferentes dispositivos

## Estructura del Proyecto

- `index.html` - Página de login
- `dashboard.html` - Panel principal del sistema
- `components/` - Componentes reutilizables
- `utils/` - Utilidades y funciones auxiliares

## Secciones del Dashboard

1. **Dashboard**: Estadísticas y gráficos principales
2. **Usuarios**: Gestión de usuarios del sistema
3. **Configuración**: Ajustes del sistema
4. **Reportes**: Generación y análisis de reportes
